# VERIFI Bill Reader

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 18.2.11.

## Angular Installation

Make sure node.js and npm are installed at [Node](https://nodejs.org/en) 

Then install Angular with `npm install -g @angular/cli`

## API Installation

Run `pip install -r requirements.txt` to install the requirements for the API.

In the pdf_processing_api folder run `uvicorn main:app --reload` to launch the API.

## Run Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## Further help

To get more help on the Angular CLI use `ng help` or go check out the [Angular CLI Overview and Command Reference](https://angular.dev/tools/cli) page.
